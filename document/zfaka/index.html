        # 常见问题

        # 管理员账号设置

        # 如何修改管理员账号及密码？

        默认账号为demo@demo.com 密码为：123456

        管理员账号需要从宝塔面板，用phpadmin打开zfaka的数据库,然后编辑其中的adminuser中的demo@demo.com替换为你想要的账号即可。

        密码可以在 ZFAKA系统后台，账户设置>账户中心>密码设置里修改。

        # 设置中心

        # 1.如何设置主题模板？

        目前支持四种主题，后期会逐步增加新主题样式。需要局部美化的，可以联系我有偿修改。

        在设置中心>配置中心>整站模板设置>选择合适的模板即可。

        • 风信子主题---列表模式

        • 朱槿主题---类淘宝模式

        • 傻蛋主题---分类列表模式

        • 迷失的选择主题---表格弹窗模式

        # 2.如何关闭用户登录？

        在设置中心>配置中心>登录开关，设置开启或关闭。

        # 3.系统邮件发送是什么？与宝塔定时发送邮件一样吗？

        在设置中心>配置中心>邮件发送，默认是系统自动发送的，也就是用户下单付款后自动发送，所以不需要去宝塔设置定时任务。如果此处关闭了，可在宝塔后台设置定时任务。

        # 4.如何修改背景图片？

        在设置中心>配置中心>前台背景图片 从网上获取图片链接，放入即可。如果是自定义图片，可存入图床后把链接放进去即可。免费图床如https://sm.ms/ 等

        # 5.如何修改导航栏背景颜色？

        在设置中心>配置中心>前台顶部菜单配色方案 ，系统自带的有黑色、藏青、墨绿、艳蓝、绿色。如需其它颜色或渐变色等，可修改内置css文件，不会的可以找人修改。

        # 6.如何替换默认Logo?

        默认logo的文件存放位置网站根目录/res/images/logo.png ，要求：格式为透明图png，标准大小135*40.制作好自己的logo后，从宝塔面板，替换logo.png文件即可。

        # 7.如何设置弹窗广告？

        在设置中心>配置中心>弹窗广告 ，主题，此处支持纯文本和html文本，注意书写格式不要出错。

        # 8.如何关闭向用户发送邮件的开关？

        在设置中心>配置中心>发送用户邮件开关 ，设置关闭或开启。

        # 9.如何设置管理员接收邮箱？

        管理员邮箱用于接收订单信息，比如用户下单时，会给管理员同步发送一份提醒邮件。

        在设置中心>配置中心>管理员邮箱

        # SMTP邮箱设置（发信设置）

        # 0.SMTP通用设置

        SMTP 的全称是“Simple Mail Transfer Protocol”，即简单邮件传输协议。它是一组用于从源地址到目的地址传输邮件的规范，通过它来控制邮件的中转方式。SMTP 协议属于 TCP/IP 协议簇，它帮助每台计算机在发送或中转信件时找到下一个目的地。SMTP 服务器就是遵循 SMTP 协议的发送邮件服务器。

        常用SMTP服务器地址

        常用邮箱 SMTP地址 端口 官方帮助文档
        腾讯QQ smtp.qq.com 465或587 mail.qq.com
        网易163 smtp.163.com 465或994 163.com
        新浪 smtp.sina.com 465或110 sina.com
        阿里云 smtp.aliyun.com 465或25 mail.aliyun.com
        雅虎 smtp.mail.yahoo.com 25 yahoo.com
        Foxmail smtp.foxmail.com 25 foxmail.com
        Hotmail smtp.live.com 587 hotmail.com
        Gmail smtp.gmail.com 587 google.com
        ..... .... ..... ......

        # 网站、数据库备份与恢复

        # 1.如何备份网站?

        打开宝塔，左侧菜单打开网站，然后对其备份。也可以通过宝塔面板自带的定时任务备份网站。恢复时只需要解压到网站目录就行。对ZFAKA程序而言，此步骤不重要，可忽略。

        # 2.如何备份数据库？

        打开宝塔，左侧菜单有数据库，然后选择 ZFAKA的数据库，进行备份即可。也可以通过宝塔自带的定时任务进行备份或通过设置定时计划的数据库备份命令行备份。

        # 3.如何升级或迁移网站或数据库？

        升级网站参考下面的教程。

        对网站而言，数据库和上传的图片是我们最终需要保留的，因为数据库包含所有账户信息、设置信息、以及商品卡密信息等，上传的图片则通过文件的方式保存。因此迁移或升级重装也很简单，只需要导出数据库和已上传的图片文件夹即可。然后新站搭建完毕后，导入旧数据库和旧的图片文件夹，一切就ok了
